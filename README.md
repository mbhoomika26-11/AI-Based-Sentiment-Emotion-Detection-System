# AI-Based-Sentiment-Emotion-Detection-System
This project is a web-based NLP application built using Streamlit that analyzes user text to detect emotions and overall sentiment. It uses TF-IDF and Logistic Regression to classify text into emotions like joy, anger, or sadness, and labels sentiment as positive, negative, or neutral.
# AI-Based Sentiment & Emotion Detection System

This project is a web-based NLP application built using Streamlit. It analyzes user text to detect emotions and sentiment.

## Features
- Emotion Detection (joy, anger, sadness, etc.)
- Sentiment Classification (positive, negative, neutral)
- Interactive UI

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLP (NLTK)

## Run Locally
pip install -r requirements.txt
streamlit run app.py
